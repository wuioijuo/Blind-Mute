# Blind & Mute

**Blind & Mute** — учебная компьютерная игра на Unity в жанре визуального симулятора ограниченного восприятия. Игрок попадает в тёмный лабиринт в стиле Backrooms, ищет фрагменты, использует способность «эхо» для обнаружения важных объектов и должен найти выход.

Проект разработан в рамках дисциплины «Основы ИТ-технологий».

## Описание проекта

Главная идея проекта — показать игровой процесс при ограниченном восприятии окружающей среды. Игрок не видит всю карту полностью: видимость ограничена туманом, затемнением, дальностью камеры и источником света около персонажа.

Для ориентации используется способность **«эхо»**, которая временно подсвечивает важные объекты: фрагменты, дверь, кнопку и выход.

## Цель игры

Найти все 5 фрагментов, активировать кнопку выхода, открыть дверь и пройти уровень.

## Основной функционал

- главное меню;
- загрузка игрового уровня;
- управление персонажем от первого лица;
- ограниченное зрение;
- туман и затемнение окружения;
- источник света у игрока;
- способность «эхо»;
- подсветка важных объектов;
- сбор фрагментов;
- взаимодействие с кнопкой и дверью;
- система подсказок;
- счётчик собранных фрагментов;
- меню паузы по клавише `ESC`;
- блокировка управления во время паузы;

## Управление

| Клавиша | Действие |
|---|---|
| `W` | движение вперёд |
| `S` | движение назад |
| `A` | движение влево |
| `D` | движение вправо |
| `Mouse` | поворот камеры |
| `E` | взаимодействие с объектами |
| `SPACE` | активация способности «эхо» |
| `ESC` | меню паузы |
| `SHIFT` | бег |

## Игровой процесс

1. Игрок запускает игру.
2. Открывается главное меню.
3. После нажатия кнопки «Начать игру» загружается уровень `Level_1`.
4. Игрок исследует тёмный лабиринт.
5. С помощью `SPACE` активируется способность «эхо».
6. Игрок находит и собирает 5 фрагментов.
7. После сбора всех фрагментов становится доступно открытие двери.
8. Игрок нажимает кнопку выхода.
9. Дверь открывается.
10. Игрок проходит через выход.
11. Загружается экран победы.

### Реализация ограниченного зрения

Ограниченное зрение реализовано несколькими способами:

включён туман;
используется затемнение и слабая освещённость уровня;
видимость ограничена источником света у игрока;
дальние участки уровня плохо различимы;
способность «эхо» временно показывает важные объекты.

Такой подход заставляет игрока ориентироваться в пространстве при неполной информации об окружающей среде.

### Меню паузы

Во время прохождения уровня можно нажать ESC.

В меню паузы доступны:
продолжить игру;
вернуться в главное меню;
выйти из игры.

При открытой паузе отключаются:
движение игрока;
поворот камеры;
взаимодействие на E;
способность «эхо» на SPACE.
